# DevBoard - Task Management API

## Overview
This is a FastAPI-based backend application for managing tasks.

## Features
- Add tasks
- View tasks
- Update tasks
- Delete tasks

## Tech Stack
- Python (FastAPI)
- Docker
- GitHub Actions

## Run Locally

cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload
